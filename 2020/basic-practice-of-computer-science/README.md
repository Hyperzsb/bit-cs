## 课程设计

实现一个部署在 Linux 平台上，实现局域网内收发邮件的全端邮件系统。

## 技术栈

- 服务端：
  - C + Socket
  - C++ (OO)
  - MySQL C API
  - Catch2
- 客户端
  - GTK3 + CSS
  - C + Socket

- **客户端：[linux-email-client](https://github.com/Hyperzsb/linux-email-server)**
